# Terrance Ford Realtor Website

Premium, conversion-focused solo realtor website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Features

- Fully responsive, mobile-first layout
- Sticky conversion header + mobile sticky contact bar
- Buyer and seller lead-capture flows
- Listings grid + listing detail template pages
- Contact API route with validation
- Local SEO metadata on every page
- RealEstateAgent and LocalBusiness schema markup
- `sitemap.xml` and `robots.txt` routes

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Copy `.env.example` to `.env.local` and update as needed.
